# git-lfs

## Install git-lfs

     git lfs install
     
## Add file name or file type to .gitattributes

     git lfs track "【file type】"

Example

     git lfs track "*.png"
     
## Add .gitattributes to git

    git add .gitattributes
    
## Check whether file type adds to git-lfs

    git lfs ls-files
